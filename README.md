#### web crawler with scrapy

## technology used

* scrapy
* python 3.6
* venv

## create virtual environment

`python3.6 -m venv virtual`

#### activate virtual

`source virtual/bin/activate`


## run

`scrapy crawl matches`

or

`python run.py`

## usage

to scrape a sport website and save the weekly game fixtures for epl.
save to json
